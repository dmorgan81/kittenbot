Every day kittenbot.io is updated with a new, AI generated image.

Each day's image name has this date format: yyyyMMdd. For example, 20230714.png.

You may also access the image specific HTML the same way: 20230714.html.

Each image has metadata about how the image was generated:

  • Model - such as foto_assisted_diffusion
  • Prompt - such as cute kitten looking at a cupcake
  • Seed - such as 4180475869

Providing these three inputs on dezgo.com should reproduce the image.

This metadata is provided in two ways:

  • The response headers x-amz-meta-model, x-amz-meta-prompt, and x-amz-meta-seed.
  • As meta tags in the HTML.
